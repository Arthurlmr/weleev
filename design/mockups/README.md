# Mockups HTML - LUMINᵉ

Ce dossier contient 8 maquettes HTML générées par IA pour l'application LUMINᵉ.

## ⚠️ Important

Les **couleurs** de ces maquettes ne correspondent **PAS** à la charte graphique.  
Référez-vous au fichier `design/brand-guidelines.md` pour la palette officielle.

## Liste des mockups

1. **conversational-profiling.html** - Profiling conversationnel par chat
2. **enhanced-property-detail-page.html** - Page détail propriété enrichie IA
3. **hybrid-feed.html** - Vue hybride liste + carte
4. **landing-page.html** - Page d'accueil marketing
5. **map-feed.html** - Vue carte interactive
6. **onboarding-flow.html** - Flux d'onboarding en 7 étapes
7. **property-feed.html** - Feed de propriétés
8. **value-saved-search-recommendations-experience.html** - Dashboard personnalisé

## Utilisation

Ouvrez les fichiers HTML directement dans un navigateur.  
Tous les styles sont inline (Tailwind CDN).

## Palette de couleurs à appliquer

**Palette Terre chaude (recommandée)**
- Primaire : `#2B2621` (noir chaud)
- Accent : `#D4A574` (or rose)
- Neutre : `#F5F1ED` (blanc cassé)
- Texte secondaire : `#6B6562` (gris chaud)
